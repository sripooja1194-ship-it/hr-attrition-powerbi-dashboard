# HR Attrition Analysis Dashboard

## Project Overview
This project analyzes employee attrition trends to identify key drivers affecting workforce retention.

## Tools Used
- Power BI
- DAX
- Excel

## Key KPIs
- Attrition Rate
- Employee Count
- Average Salary
- Department-wise Attrition
